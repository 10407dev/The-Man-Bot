### Installing
- Clone the repository
- Run `npm install`

### Starting
- Create `.env` file in repository root
- Fill that file like so: 
```
TOKEN=TOKENHERE
CLIENT_ID=CLIENTIDHERE
TESTING_GUILD_ID=TESTINGGUILDIDHERE
# TESTING_GUILD_ID is required for dev commands to be deployed
```

- Run `npm run deploy`
- Run `npm run start`
